# Blueprint: SEO Optimized Hunter x Hunter Fan Site

## 1. Project Overview

This project aims to build a multi-page static website about the anime/manga "Hunter x Hunter". The primary goal is to create a site that is highly optimized for search engines (SEO) by strictly following the guidelines provided by Google and Naver.

The site will feature unique content, a clean and modern design, a responsive layout, and a logical structure that is easy for both users and search engine crawlers to navigate.

## 2. Core SEO Strategy

- **Content:** Unique and valuable content related to "Hunter x Hunter" on every page.
- **Crawlability:** A `robots.txt` file to allow all search engines to crawl the site, and a `sitemap.xml` to provide a clear map of all pages.
- **Metadata:** Unique `<title>` and `<meta name="description">` tags for each page, plus Open Graph tags for social media sharing.
- **Semantic HTML:** Correct use of tags like `<header>`, `<footer>`, `<main>`, `<nav>`, `<article>`, and `<h1>`-`<h6>` to define content structure.
- **URL Structure:** Clean, descriptive, and hierarchical URLs.
- **Image Optimization:** Meaningful `alt` attributes for all images.
- **Internal Linking:** Clear navigation and contextual links between relevant pages.

## 3. Design & User Experience (UX)

- **Visuals:** A modern design with a color scheme and typography inspired by the "Hunter x Hunter" universe.
- **Responsiveness:** A fully responsive design that works on all devices.
- **Usability:** Intuitive navigation and a clear layout for easy access to information.

4.  **[completed]** **Header & Footer:** Create reusable header and footer structures with navigation and social links, to be included on all pages.
5.  **[completed]** **Content Pages:** Build out the individual pages (`about.html`, `articles.html`, `article-1.html`, etc.) with placeholder content and proper SEO tags.
6.  **[completed]** **Sitemap Generation:** Populate `sitemap.xml` with all the site's URLs.
7.  **[completed]** **Final Review:** Perform a final check for broken links, accessibility issues, and SEO best practices.
